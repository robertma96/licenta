Welcome

The first thing the app does is to identify the faces present in a photo. After that, it will apply the style of a famous picture selected by you.

This is an application used for styling images (applying a filter). First step is to upload a photo that contains faces.
After that you will need to select painting that you like the most (this will be the filter you selected). After that you have to select one of the two modes:
Style only the faces OR Style only the background and not the faces

So to recap:

  1. Create an account or log in or Sign in
  2. Upload a photo that contains faces
  3. Select a famous painting
  4. Select if you want only your face to be styled or if you the background to be styled, but not your face
  5. Click on the Get Started to start styling photos
